## Project Overview
A comprehensive data engineering pipeline for Egyptian Exchange (EGX) market data processing and analytics.

## Architecture
- **Extract:** Kaggle (batch) + EGX Web Scraping (real-time)
- **Bronze Layer:** AWS S3 + Apache Kafka
- **Silver Layer:** DBT + Apache Spark
- **Gold Layer:** Snowflake Data Warehouse
- **Orchestration:** Apache Airflow
- **Containerization:** Docker
- **Visualization:** Dashboards
